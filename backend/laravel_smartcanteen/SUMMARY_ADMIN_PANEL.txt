✅ FITUR YANG SUDAH DIBUAT:

1. 📊 DASHBOARD
   - Statistik: Total Users, Categories, Menus, Orders
   - Widget: Pending Orders
   - URL: /admin/dashboard

2. 👥 USER MANAGEMENT (CRUD)
   - Create: Tambah user baru dengan role (admin/pelanggan)
   - Read: List & detail users dengan pagination
   - Update: Edit nama, email, password, role
   - Delete: Hapus user
   - URL: /admin/users

3. 🏷️ CATEGORY MANAGEMENT (CRUD)
   - Create: Tambah kategori baru
   - Read: List kategori dengan counter jumlah menu
   - Update: Edit nama kategori
   - Delete: Hapus kategori (dengan validasi)
   - URL: /admin/categories

4. 🍔 MENU MANAGEMENT (CRUD)
   - Create: Tambah menu dengan upload gambar
   - Read: List menu dengan gambar & indikator stok
   - Update: Edit menu & ganti gambar
   - Delete: Hapus menu & gambar
   - Field: Kategori, Nama, Harga, Stok, Gambar, Deskripsi
   - URL: /admin/menus

5. 🛒 ORDER MANAGEMENT
   - Create: Buat order baru
   - Read: List & detail orders
   - Update: Update status order (pending/diproses/selesai/dibatalkan)
   - Delete: Hapus order
   - URL: /admin/orders

6. 📦 ORDER DETAIL MANAGEMENT
   - Create: Tambah item ke order
   - Delete: Hapus item dari order
   - Auto calculate: Total harga & subtotal
   - Auto manage: Stok menu
   - URL: /admin/orders/{id}/details
